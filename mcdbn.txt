MCDBControl.AbortOperation
Abort Burn, BuildISOImage, BurnISOImage or VerifyDisc operation.

Visual Basic (Declaration)

Public Sub AbortOperation()

This method is used to abort the on-going operation. It instantly stops writing data to the disc and leaves the disc as-is. This method should be used with care, as the disc will be left in an open state.

Note: This method will not abort an EraseDisc operation.

MCDBControl.BlocksWritten

Visual Basic (Declaration)

Public Function BlocksWritten() as Integer

Returns the number of Blocks (1 Block = 2048 Bytes) written to the disc, this method is only available during Burn, BuildISOImage, BurnISOImage Operation.

Example
Private Sub Timer1_Elapsed(ByVal sender As System.Object, ByVal e As System.Timers.ElapsedEventArgs) Handles Timer1.Elapsed
Const CS_IDLE As Integer = 0
Const CS_WRITING As Integer = 1
Const CS_ERASING As Integer = 2
Const CS_BACKGROUNDFORMAT As Integer = 3
Const CS_CLOSINGTRACK As Integer = 4
Const CS_PREPARING As Integer = 5
Const CS_ABORTING As Integer = 6
Const CS_IMPORTINGSESSION As Integer = 7
Const CS_BURNERROR_ABORTING As Integer = 8
Const CS_VERIFYING As Integer = 9
Const CS_CLOSINGDISC As Integer = 10
Const CS_GEN_SM_REFERENCES As Integer = 11
If mcdb.State = CS_IDLE Then
msg.Text = "Idle"
ElseIf mcdb.State = CS_ERASING Then
msg.Text = "Erasing ..."
ElseIf mcdb.State = CS_CLOSINGTRACK Then
msg.Text = "Closing Track ..."
ElseIf mcdb.State = CS_PREPARING Then
msg.Text = "Preparing ..."
ElseIf mcdb.State = CS_ABORTING Then
msg.Text = "Aborting ..."
ElseIf mcdb.State = CS_BURNERROR_ABORTING Then
msg.Text = "Aborting ..."
ElseIf mcdb.State = CS_GEN_SM_REFERENCES Then
pb.Maximum = mcdb.SmartReferencesSize
pb.Value = mcdb.SmartReferencesProgress
lSize.Text = Trim(Str((pb.Value * 0.002))) + "/" + Trim(Str(pb.Maximum * 0.002))
msg.Text = "Gen Sm References ..."
ElseIf mcdb.State = CS_WRITING Then
pb.Maximum = mcdb.ImageSize
pb.Value = mcdb.BlocksWritten
lSize.Text = Trim(Str((pb.Value * 0.002))) + "/" + Trim(Str(pb.Maximum * 0.002)) '& "( at " & mcdb.DeviceOperationSpeed & " KB/s"
pbDB.Maximum = mcdb.DeviceBufferSize
pbDB.Value = mcdb.DeviceBufferPosition
pbSB.Maximum = mcdb.CacheSize
pbSB.Value = mcdb.CachePosition
msg.Text = "Writing ..."
ElseIf mcdb.State = CS_VERIFYING Then
pb.Maximum = mcdb.VerifyDataSize
pb.Value = mcdb.VerifyProgress
lSize.Text = Trim(Str((pb.Value * 0.002))) + "/" + Trim(Str(pb.Maximum * 0.002))
msg.Text = "Verifying ..."
End If
End Sub

MCDBControl.Burn
Starts the process of burning files/directories on Disc.

Visual Basic (Declaration)

Public Function Burn() as Boolean

This method begins the disc writing operation. Files/Directories, to be burnt on a media, should be added to disc layout, prior to calling this method. This operation is asynchronous i.e. this method will return a TRUE or a FALSE immediately, as soon as the writing process is initiated. An OnWriteDone event will be triggered when write process is finished (whether successfully or not).

Example
mcdb.ClearAll()
mcdb.InsertFile("\", "c:\boot.ini")
mcdb.Prepare()
If mcdb.Burn() Then
AppendLog(">>> Now Burning ...")
Else
AppendLog(">>> Burn() Failed")
End If

msg.Text = "Idle"
ElseIf mcdb.State = CS_ERASING Then
msg.Text = "Erasing ..."
ElseIf mcdb.State = CS_CLOSINGTRACK Then
msg.Text = "Closing Track ..."
ElseIf mcdb.State = CS_PREPARING Then
msg.Text = "Preparing ..."
